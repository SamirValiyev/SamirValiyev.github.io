C#

C# (C Sharp) Microsoft tərəfindən yaradılmış obyekt yönümlü (object-oriented) proqramlaşdırma dilidir. C#, .NET platformasında tətbiqlər hazırlamaq üçün geniş şəkildə istifadə olunur və müasir proqramlaşdırma dillərindən biridir. C++, Java kimi dillərə bənzər sintaksisə malikdir, lakin daha sadə və istifadəsi asan bir dil olaraq qəbul edilir.

MSSql

Microsoft SQL Server (MSSQL), Microsoft tərəfindən yaradılmış bir məlumat bazası idarəetmə sistemidir (DBMS). Bu sistem, müxtəlif miqyaslı tətbiqlərin məlumatlarını saxlamaq, idarə etmək və analiz etmək üçün istifadə olunur. MSSQL, xüsusilə korporativ səviyyəli tətbiqlər və geniş məlumat bazaları üçün geniş istifadə olunan bir həll yoludur.

EF core

Entity Framework Core (EF Core), Microsoft tərəfindən yaradılmış ORM (Object-Relational Mapping) vasitəsidir. ORM, obyekt yönümlü proqramlaşdırma dillərində yazılmış kodla SQL məlumat bazaları arasında körpü yaradır. Bu, yəni proqramçıların SQL yazmadan, C# kimi proqramlaşdırma dilləri ilə məlumat bazası əməliyyatlarını yerinə yetirməsini təmin edir. EF Core .NET platformasında istifadə olunur və ən son versiyası ilə çoxplatformalı dəstək təqdim edir (Windows, Linux, macOS).

ASP.Net core MVC

ASP.NET Core MVC, Microsoft-un yaratdığı çoxplatformalı bir veb proqramlaşdırma çərçivəsidir və Model-View-Controller (MVC) arxitekturasına əsaslanır. Bu çərçivə veb tətbiqlərinin yaradılması üçün geniş imkanlar təqdim edir və həm sadə, həm də mürəkkəb veb saytların və API-lərin inkişaf etdirilməsi üçün uyğundur. ASP.NET Core MVC, .NET Core platformasında işləyir və buna görə çoxplatformalı dəstək verir, yəni Windows, Linux və macOS əməliyyat sistemlərində istifadə oluna bilər.

ASP.Net core Api

ASP.NET Core API, Microsoft-un RESTful (Representational State Transfer) servislər yaratmaq üçün təklif etdiyi bir çərçivədir. Bu çərçivə, server tərəfli tətbiqlərin hazırlanmasında geniş istifadə olunur və JSON, XML kimi formatlarla məlumatların serverdən müştəri tərəfə ötürülməsini təmin edir. RESTful API-lər müasir veb və mobil tətbiqlərin arxasında duran əsas texnologiyalardan biridir və məlumatların mübadiləsini asanlaşdırır.
